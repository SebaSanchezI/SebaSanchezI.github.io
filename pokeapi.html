Pokeapi

Pokeapi es un proyecto diseñado y desarrollado como Single Page Application dentro del marco del bootcamp de soyHenry. La app se encuentra desarrollada con React, Redux y CSS puro para la parte del Front y con Node JS, Express, Sequelize y Postgres para el Back.

  • Combina el consumo de datos a una API externa (pokeapi.co) y una base de datos local.
  • Ofrece como característica extra un CRUD de personajes.
  • Los datos de estos nuevos personajes son los que consume de la base de datos local.
  • Presenta un cache de búsqueda, filtros, ordenamientos y un formulario controlado.
