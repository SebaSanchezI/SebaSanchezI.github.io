Admin Stores

Admin Stores es una aplicacion SPA (Single Page Application), que surge como solucion a un challenge enviado por una empresa. El objetivo de la misma es administrar distintas tiendas, las cuales poseen usuarios con diferentes roles (cajero y supervisor). Diseñada y desarrollada con React, Redux, Material UI para el front y Express, Sequelize y Postgres en el Back. Utilizacion de custom Hooks y Context. Para graficar los datos se utilizo React Chart Js y Sweetalert para customizar las alertas. La API se encuentra documentada con Swagger.
Algunas de sus características son:

  • Login de Usuarios.
  • Autenticación del login mediante JWT (Json Web Token).
  • Tablas de usuarios con sus datos.
  • Graficos Dinamicos de estaditicas de usuarios por tienda.
  • CRUD de usuarios.
  • Encriptscion de password.
  • Filtros y busqueda de usuarios.
  • Base de datos relacional.
